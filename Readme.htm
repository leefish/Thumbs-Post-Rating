Overview:

This plugin will add a thumbs up / thumbs down rating system on every individual posts.

Just like the one on the YouTube comment.

 

Features:

·         Non-obstructive AJAX (send the rating and update the result instantly without refreshing the page)

·         Language file support

·         Able to set which usergroup allowed to rate

·         Able to set which forum to be excluded to have the post rating function

·         Able to enable/disable self-rating

·         Compatible with both normal and classic postbit

 

Technical information:

Number of files                : 6

Database changes           : Added 1 new table

                                                  Added 2 columns on an existing table

Template changes           : Modified 3 templates

SQL queries                        : One per post on posts display

 

Installation:

1.       Upload all the files in the "Upload" folder to the root of your forum.

2.       In ACP Plugins section, click "Install & Activate" next to the Thumbs Post Rating.

 

Uninstallation:

In ACP Plugins section, click "Uninstall" next to the Thumbs Post Rating.

 

Upgrading from previous version:

1.       In ACP Plugins section, click "Deactivate" next to the Thumbs Post Rating.

2.       Upload all the files in the "Upload" folder to the root of your forum. If required, choose to replace any old files.

3.       In ACP Plugins section, click "Activate" next to the Thumbs Post Rating.
